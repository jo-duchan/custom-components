## Overview
custom-components는 Pragmatism Design System을 사용하여 핵심 코어가 되는 컬러, 타이포그래피부터
다양한 커스텀 컴포넌트들을 만들어 디자인 시스템을 구축하는 사이드 프로젝트입니다.

Pragmatism Design System은 피그마 커뮤니티 OP Labs으로부터 디자인되었습니다.
